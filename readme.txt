Отправка смс уведомление на телефон через смс сервис рассылок sms.ru.

== Description ==

**Данный плагин отправляет смс уведомления на телефон администратора или клиента при отправке данных с формы Contact Form 7, через сервис [sms.ru](http://gosend.sms.ru)**

== Installation ==

1. Upload `cf7smsru` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
